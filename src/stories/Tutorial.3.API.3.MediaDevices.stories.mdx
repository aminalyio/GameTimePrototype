# MediaDevices. Public method summary

The MediaDevices interface provides access to connected media input devices like cameras and microphones.

## getMediaDevices()
`getMediaDevices()` - Returns the list of available for using media devices

## setAudioConstraints(deviceId)
`setAudioConstraints(deviceId)` - Selects the audio device for the session

## setVideoConstraints(deviceId)
`setVideoConstraints(deviceId)` - Selects the video device for the session

## doPreview(constraints)
`doPreview(constraints)` - Returns a preview stream for session

## changeMediaDevice(deviceId, type, selector)
`changeMediaDevice(deviceId, type, selector)` - Allows the user to change media device

* deviceId - id of selected device provided by getMediaDevices() method
* type - type of device to change. Possible values are: ['camera', 'microphone', 'speaker']
* querySelector for video element to replace a stream object
