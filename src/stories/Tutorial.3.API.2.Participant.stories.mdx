# Public fields

`participantId` - Unique identifier of a participant


`participantName` - Name of participant


`isLocal` - Identifies whether the participant is a local one


`stream` - Participant Media Stream - Read here for more information


`streamConstraints` - An object with two fields:
{audio: boolean, video: boolean}
* Example: 
  ```js
  {audio: false, video: true}
  ```
  That means a stream of participant consists only of video tracks


`settings` - An object with two fields:\
* Example: 
  ```js
  {mutedAudio: boolean, mutedVideo: boolean}
  ```
  Those fields contain the state of the Audio and Video of the participant
